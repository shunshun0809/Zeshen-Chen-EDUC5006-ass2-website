# 📐 Maths at Home: Measuring Our World

## 项目简介 / Project Overview

这是一个为二年级学生设计的交互式数学远程学习网站，主题为"测量我们的世界"。该网站在洪水期间为学生提供在家学习测量概念的完整学习体验。

An interactive mathematics remote learning website designed for Year 2 students, themed "Measuring Our World". This website provides a complete learning experience for students to learn measurement concepts at home during flood disruptions.

## ✨ 主要功能 / Key Features

### 📚 多页面学习结构 / Multi-page Learning Structure
- **首页 (Home)**: 欢迎信息和教师视频消息
- **第1页 - 学习规则 (Learning the Rules)**: 通过视频和互动游戏学习非标准单位测量
- **第2页 - 开始测量 (Let's Get Measuring)**: 实践活动和照片分享
- **第3页 - 故事时间 (Story Time)**: 音频故事和反思问卷
- **第4页 - 数据图表 (Graphing Our Data)**: 学习制作条形图和Kahoot测验
- **资源页 (Resources)**: 所有使用工具的链接和成就证书

### 🎨 交互式元素 / Interactive Elements
- Loom 视频嵌入 (教师消息)
- YouTube 教学视频
- Wordwall 互动分类游戏
- Padlet 数字公告板 (学生作品分享)
- 音频播放器 (故事朗读)
- Google Forms 问卷调查
- Kahoot 互动测验
- PDF 文档查看器

### 💫 用户体验设计 / UX Design
- 响应式设计，支持移动设备和桌面
- 色彩丰富的渐变背景
- 平滑的页面过渡动画
- 进度指示器
- 友好的表情符号和视觉元素
- 悬停效果和动画

## 🎯 学习目标 / Learning Objectives

1. **理解测量概念**: 使用非正式单位进行测量
2. **掌握测量规则**:
   - 使用相同尺寸的单位
   - 端对端测量
   - 不留间隙
   - 直线排列，不重叠
3. **数据收集**: 使用日常物品测量家中物品
4. **数据可视化**: 创建和理解条形图
5. **批判性思维**: 通过故事和问卷培养分析能力

## 🚀 使用方法 / How to Use

### 本地运行 / Local Setup
1. 克隆或下载此仓库
2. 直接在浏览器中打开 `index.html` 文件
3. 无需服务器或额外配置

### 在线部署 / Online Deployment
- 可以部署到 GitHub Pages、Netlify、Vercel 等静态网站托管平台
- 确保 `resources/` 文件夹中的所有资源文件都已上传
